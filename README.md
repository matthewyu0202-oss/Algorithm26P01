Brute Force 알고리즘: 모든 경우의 수를 탐색하여 답을 찾는다.
각 문제 파일(`solve1.py`, `solve2.py`)을 열고, 조건에 맞게 함수를 완성하여 제출해 주세요.

---

### 문제 1: 조건에 맞는 수의 합 구하기 (solve1.py)
**[문제 설명]**
주어진 정수 리스트 안에서 **3의 배수이면서 홀수인 수**만 골라 그 합을 반환하는 함수를 작성하세요. 
만약 조건에 맞는 수가 단 하나도 없다면 `0`을 반환해야 합니다. 

**[제한 사항]**
* `numbers` 리스트의 길이는 1 이상 1,000 이하입니다.
* 리스트의 각 원소는 1 이상 10,000 이하의 정수입니다.

**[입출력 예시]**
| 입력 (`numbers`) | 출력 (반환값) | 설명 |
| :--- | :--- | :--- |
| `[1, 2, 3, 4, 5, 6, 7, 8, 9]` | `12` | 3, 9가 조건에 해당. 3 + 9 = 12 |
| `[2, 4, 6, 8, 10]` | `0` | 3의 배수 중 홀수가 없으므로 0 |

### 문제 2: 보안 카드 비밀번호 조합 (solve2.py)
**[문제 설명]**
새로운 보안 시스템의 비밀번호 형식이 문자열로 주어집니다. 
형식은 문자를 의미하는 `'C'` (Character, 알파벳 A-Z 26개)와 숫자를 의미하는 `'N'` (Number, 0-9 10개)으로 구성됩니다.
보안을 위해 **같은 형식(문자 또는 숫자)이 연속으로 두 번 이상 등장할 경우, 바로 앞자리와 똑같은 문자/숫자는 사용할 수 없습니다.** 주어진 형식 `format_str`로 만들 수 있는 가능한 모든 비밀번호의 개수를 반환하는 함수를 작성하세요. (BOJ 16968번 차량 번호판 1 응용)

**[제한 사항]**
* `format_str`의 길이는 1 이상 4 이하이며, `'C'`와 `'N'`으로만 이루어져 있습니다.

**[입출력 예시]**
| 입력 (`format_str`) | 출력 (반환값) | 설명 |
| :--- | :--- | :--- |
| `"NN"` | `90` | 첫 자리는 10개, 두 번째 자리는 앞자리와 달라야 하므로 9개 (10 * 9 = 90) |
| `"CN"` | `260` | 첫 자리 문자 26개, 두 번째 자리 숫자 10개 (26 * 10 = 260) |
| `"CCN"` | `6500` | C(26) * C(25) * N(10) = 6500 |
